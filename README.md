# Coder-Square

Hacker News Backend APIs

## API

**Auth:**

```
/v1/signup [POST]
/v1/signin [POST]
```

**Posts:**

```
/v1/posts [POST]
/v1/posts [GET]
/v1/posts/:postId [DELETE]
```

**Likes:**

```
/v1/likes/:postId [POST]
/v1/likes/:postId [GET]
```

**Comments:**

```
/v1/comments/:postId [POST]
/v1/comments/:postId [GET]
/v1/comments/:commentId [DELETE]
```
